# Chrome-Extension

Lightweight frontend for a Chrome extension. This repository contains the UI, assets, and build tooling used to produce an unpacked Chrome extension (Manifest V3).

## Prerequisites

- Chrome or Chromium-based browser

## Quick start

1. Clone the repo

   git clone <repo-url>
   cd Chrome-Extension

## Load extension in Chrome (unpacked)

1. Open chrome://extensions
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the output folder
